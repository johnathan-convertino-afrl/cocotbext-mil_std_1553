Architecture

Please see for more information.

MILSTD1553Source tests mil-std-1553 receive devices. This is uses the machester encoder library in python, then this is bit banged out using timers. The sync is sent based upon which write command is used. write_cmd will write the data and use a command sync out. write_data will write the data and use a data sync out.

MILSTD1553Sink tests mil-std-1553 transmit devices. This is uses the machester decoder library in python, then this is bit banged in using timers. Once a sync is identified the data is put in one of two queues. The data queue contains data that came from a message that started with a data sync. The command queue contains data that came from a message that started with a command/status sync.
